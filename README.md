PasswordForge – Генератор паролей с оценкой сложности на 7 языках
Создавай стойкие пароли и получай моментальную оценку их надёжности.
Поддержка 7 языков, расширенные настройки, проверка на словари, исключение неоднозначных символов, генерация произносимых паролей и экспорт в файл.

🚀 Возможности
Генерация паролей с настраиваемыми параметрами:

Длина (по умолчанию 16).

Включение/исключение: заглавные, строчные, цифры, спецсимволы.

Исключение неоднозначных символов (O0o1lI и т.п.).

Режим «произносимые» – создание паролей по слогам (легче запомнить).

Оценка сложности:

Вычисление энтропии (бит).

Примерное время взлома перебором (в секундах, годах и т.д.).

Проверка на наличие в словаре распространённых паролей (встроенный словарь топ-1000).

Продвинутые фичи:

Генерация нескольких паролей за раз (-n).

Минимальная энтропия – генерация повторяется, пока не будет достигнута.

Вывод в файл (-o).

Цветной вывод (где поддерживается).

Единый интерфейс для всех языков – легко переключаться.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Доступные опции
Опция	Описание
-l, --length	Длина пароля (по умолчанию 16)
-c, --count	Количество паролей для генерации (по умолчанию 1)
-u, --uppercase	Включать заглавные буквы (по умолчанию да)
-L, --lowercase	Включать строчные буквы (по умолчанию да)
-d, --digits	Включать цифры (по умолчанию да)
-s, --special	Включать спецсимволы (по умолчанию да)
-e, --exclude	Исключать неоднозначные символы (O0o1lI и т.п.)
-p, --pronounce	Генерировать произносимые пароли (игнорирует другие наборы)
-m, --min-entropy	Минимальная энтропия (бит) – повторять генерацию до достижения
-q, --quiet	Не выводить оценку сложности (только пароль)
-o, --output	Сохранить результат в файл
-h, --help	Справка
Примеры
bash
# Сгенерировать пароль длиной 20 со всеми символами
python password_forge.py -l 20

# 5 паролей без спецсимволов, с исключением неоднозначных
python password_forge.py -c 5 -s false -e

# Произносимый пароль из 12 символов
python password_forge.py -p -l 12

# Пароль с минимальной энтропией 80 бит
python password_forge.py -m 80

# Сохранить результат в файл
python password_forge.py -o passwords.txt

🧠 Архитектура
Каждая программа построена по единому шаблону:

Парсинг аргументов.

Генерация пароля (или нескольких) с учётом настроек.

Оценка сложности (энтропия, время взлома, проверка словаря).

Вывод результатов (с цветом, если разрешено).

Запись в файл при необходимости.

Код оптимизирован для производительности и читаемости.

✨ Дополнительные фичи
Встроенный словарь топ-1000 самых распространённых паролей для проверки.

Расчёт времени взлома в зависимости от скорости перебора (по умолчанию 10⁹ паролей/с).

Генерация произносимых паролей – чередование согласных и гласных, чтобы было легко запомнить.

Автоматическое повышение сложности, если энтропия ниже заданной.

Цветной вывод – зелёный для высокой стойкости, жёлтый для средней, красный для слабой.
